Preguntas del laboratorio

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Algunas de las principaes diferencias es que JavaScript es un lenguaje que es interpretado, mientras que Java es uno compilado, esto quiere decir que JavaScript puede integrarse directamente en páginas HTML y se interpretan por el navegador y Java tiene que compilarse en un archivo especial. Tambien es el hecho de que Java es un lenguaje orientado a objetos, mientras que JavaScript es un lenguaje que esta más orientado a eventos.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

1- Date.now() - Regresa el valor de los milisegundos transcurridos desde el 1 de enero de 1970 hasta el momento actual.

2- Date.parse() - Recibe una cadena indicando una fecha y regresa los milisegundos transcurridos entre la fecha y el 1 de enero de 1970

3- Date.UTC() - Tiene una función similar a la anterior pero los parametros deben ser más para tener una mayor precisión.

4- Date.getFullYear() - Regresa el año actual.

5- Date.getMinute() - Regresa el minuto actual.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

1- concat - Concatena uno o más arreglos y los regresa.

2- sort - Ordena los elementos de un array.

3- push - Añade un nuevo elemento al final del arreglo y regresa su nuevo tamaño.

4- pop - Remueve un elemento del arreglo y regresa dicho elemento.

5- forEach - Llama a una función determinada por cada elemento del arreglo.

¿Cómo se declara una variable con alcance local dentro de una función?

Una variable con alcance local se declara con la palabra "let" antes de la misma y dentro de la función.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Esto tiene varias implicaciones como el hecho de que esta cambiara a su valor al anterior u original al salir de la función si no se imprime o utiliza durante la función en cuestión.
